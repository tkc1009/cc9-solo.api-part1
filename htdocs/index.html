AI Pokemon Battle API

This was created during my time as a student at Code Chrysalis

Beginning

Pokemon Battle is 6 on 6.
Each player can have 6 pokemons.
Each pokemon can have 4 moves.

Purpose

You can find out the best action for move or change when you battle with rival.

Usage

AI Pokemon Battle API

New User:
mutation { insertPlayer(player: { playerName: "blue" }) { playerId playerName } }
Select User:
query { selectPlayer(player: { playerId: ${Your Id} }) { playerId playerName } }
Update User:
mutation { updatePlayer(player: { playerId: ${Your Id}, playerName: ${Your New Name}} ) { playerId playerName } }
Delete User:
mutation { deletePlayer(player: { playerId: ${Your Id}} ) }
New Your Pokemon:
mutation { insertPartyMember(partyMember: { playerId: ${Your ID}, pokemonId: ${Your Pokemon ID or Name} }) { playerId partyMemberId pokemonId pokemonName } }
Select Your Pokemon:
query { selectPartyMember(partyMember: { playerId: ${Your Id} }) { playerId partyMemberId pokemonId pokemonName } }
Update Your Pokemon:
mutation { updatePartyMember(partyMember: { playerId: ${Your Id}, partyMemberId: ${Your Party Member Id}, pokemonName: ${Your Pokemon ID or Name} }) { playerId partyMemberId pokemonId pokemonName } }
Delete User:
mutation { deletePartyMember(partyMember: { playerId: ${Your Id}, partyMemberId: ${Your Party Member Id} }) }
